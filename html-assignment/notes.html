Notes

a
Defines a hyperlink, which is used to link from one page to another.
img
Defines an image in an HTML page. The 'src' attribute specifies the path to the image.
ul
Creates an unordered list. List items are marked with bullets.
ol
Creates an ordered list. List items are marked with numbers or letters.
div
Defines a division or a section in an HTML document. Mainly used for styling with CSS.
